# public-transport-tracker
Real-Time Public Transport Tracking System for small cities using GPS and web technologies. It provides live bus location, route details, and estimated time of arrival (ETA) through a user-friendly interface. Helps reduce waiting time, improve travel planning, and enhances overall public transport efficiency.
# Real-Time Public Transport Tracking System

This project tracks buses in real-time using GPS and displays location and ETA.

## Features
- Live bus tracking
- ETA calculation
- Web interface
